PCBox

A PC emulator focused on accuracy and hardware preservation.


What can I do with PCBox?

  • Learn how computers work at a low level by exploring the source code
  • Run certain demos that donâ€™t work on much else than real hardware (Like 8088MPH, for instance)
  • Run old games or applications, on emulated hardware that they were designed for
  • Intel Pentium 3 emulation
  • Experimental SSE2 and SSE3 emulation
  • Atari PC 4, VIA C3 and Intel i815 (From "feature/ich2" branch of 86Box) emulation
  • Experimental coreboot support on some emulated boards

    • And more!

System requirements

Minimum
  • CPU: Intel Core 2 Duo
  • OS: Windows 7 or modern Linux
  • RAM: 4 GB
Recommended
  • CPU: AMD Ryzen 7 9800X3D or Intel Core i5 12600K
  • OS: Windows 11 or modern Linux
  • RAM: 16 GB

Planned improvements for the future

  • Intel Pentium 4 emulation (including SSE2)
  • AMD Athlon and AMD Athlon XP emulation
  • More accurate Intel Pentium 3 speeds
  • New (optional) dynamic recompiler that sacrifices timing accuracy for speed
  • x86_64 emulation (far future)
